How does JavaScript work?

JavaScript is a client-side scripting language and one of the most used scripting languages. At present, most of the modern web browsers support javascript and have their javascript engines. Google Chrome has its own Javascript engine called V8. JavaScript was initially developed as a fronted scripting language to fulfil the lacking dynamic behavior of the web. A javascript engine is basically a software that executes JavaScript code. Now, V8 is the most used javascript engine .

Difference between local storage and session storage ?

Local storage and session storage provide a private area for data.

Local storage hasn't expiration date. Session storage expires on tab close.

Local storage has 10mb of holding capacity and Session storage has 5mb of holding capacity

Local storage and session storage are editable by users

Both can be stored on browser only.
